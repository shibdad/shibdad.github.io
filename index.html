Healthcare ML · XGBoost · SHAP · Epic Cosmos

                        
HIV Risk Prediction Model

                        

                            Built and deployed an XGBoost model trained on Epic Cosmos data to proactively identify
                            patients at elevated HIV risk — enabling clinical outreach before symptoms appear.
                            Jefferson Health was the first organization in the Epic ecosystem to
                            translate a Cosmos research model through Slate development into a live Nebula
                            production deployment with an operational OPA.
                        

                        

                            
0.805
AUC — target departments

                            
114K+
Patients scored

                            
~1,789
High-risk patients flagged

                        

                        

                            

                                Fairness-aware design
                                Deliberately removed race and SVI features to prevent disproportionate flagging, accepting a 0.053 AUC trade-off for ethical deployment
                            

                            

                                SHAP in production
                                TreeExplainer integrated into Epic PMA — clinicians see per-patient feature contributions directly in Hyperspace
                            

                            

                                Deployment pipeline
                                Cosmos → Slate Docker container → Nebula production, solving version conflicts across scikit-learn, XGBoost, and pickle serialization

NLP · LLM · Epic PMA

                    
Valvular Disease NLP Pipeline

                    

                        Production GPT-4o inference pipeline that classifies echocardiogram impressions into
                        multi-class valvular severity labels (none / trace-mild / moderate / severe) when
                        structured documentation is absent. Migrated from Epic PMA to Microsoft Fabric using
                        Azure OpenAI and Spark SQL for scale beyond the 15-minute RWB timeout limit.
                    

                    

                        GPT-4oAzure OpenAIMicrosoft FabricSpark SQL
                    

                


                

                    
NLP · Transformers · Research

                    
Brain Edema Detection In Research

                    

                        Exploring lightweight transformer alternatives (<5GB, CPU-only) to replace high-cost
                        LLM inference for brain edema classification from radiology impressions — targeting
                        Epic Nebula deployment constraints. Current approach uses GPT-4o via Epic's AI Services
                        framework; evaluating fine-tuned DistilBERT and model distillation strategies.
                        Labeled 82K reports for training data.
                    

                    

                        DistilBERTHugging FacePyTorchGPT-4oEpic PMA
                    

                


                

                    
LLM · Clinical NLP · Epic Workflow API

                    
Clinical Note Summarizer

                    

                        GPT-4o-powered Epic Workflow API model that summarizes clinical notes on demand,
                        surfacing output directly in the patient list via a StoreStructuredText print group
                        in Hyperspace. Built using Epic's native cognitive inference framework
                        (epic.io.workflow_integrations) extracted from Slate Docker.

RAG · Clinical Guidelines · Care Gaps

                    
Care Gap Detection (RAG)

                    

                        Retrieval-augmented pipeline that grounds an LLM in clinical guideline text to
                        surface open care gaps for a patient — retrieving the relevant guidance, then
                        generating a recommendation tied back to its source so clinicians can verify it
